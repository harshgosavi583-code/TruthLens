🤖 About TruthLens

TruthLens is an AI-powered Fake News Detection System developed to identify misleading news articles using Machine Learning.

Technologies Used

  • HTML
  • CSS
  • Python
  • Flask
  • Machine Learning
  • SQLite

How TruthLens Works

  1. Paste a News URL or News Article.
  2. TruthLens extracts the news content.
  3. The AI model analyzes the article using Machine Learning.
  4. The system predicts whether the news is Real or Fake.
  5. The prediction and confidence score are saved in Prediction History.
